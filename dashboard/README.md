# 第四阶段：仪表盘卡片

## 目标

创建 Lovelace Custom Card，复刻虚拟室内机界面和弹窗交互。

## 卡面设计

### 主卡片（门口机列表）

```
┌──────────────────────────────────┐
│ 🏠 云海湾门禁系统            ⚙️ │
├──────────────────────────────────┤
│ 🟢 1号机 / 1层 车库             │
│ 🟢 2号机 / 2层 花园             │
│ ⚫ 3号机 / -1层 车库             │
│ ...                              │
└──────────────────────────────────┘
```

- 状态指示：🟢=在线、⚫=离线
- 点击某号机卡片 → 打开监控弹窗

### 呼叫弹窗（呼入自动弹出）

```
┌──────────────────────────────────┐
│ 1号机 正在呼叫           ✕     │
├──────────────────────────────────┤
│                                  │
│        [视频画面区域]            │
│                                  │
├──────────────────────────────────┤
│   [🔓解锁]   [📞接听]   [挂断]  │
└──────────────────────────────────┘
```

- 自动弹出（监听 call_started 事件）
- 视频显示、接听、解锁按钮

## 实现计划

1. **doorlock-card.js** — 自定义卡片的 JavaScript 模块
2. **doorlock-card-editor.js** — 卡片的配置编辑器（可选）
3. **doorlock-card.json** — HACS 资源声明

## 技术方案

- 使用 LitElement（Web Component）开发
- 调用 HA API（websocket 或 REST）获取状态
- 调用 HA 服务 `uppercoast_doorlock.unlock/answer/hangup`
- 通过 HA 事件监听呼入

## 依赖

- Home Assistant 前端（内置 LitElement）
- uppercoast_doorlock 集成（需先安装）
