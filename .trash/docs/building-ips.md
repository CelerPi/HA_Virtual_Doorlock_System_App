# 楼栋 IP 规则

这个文件记录当前已整理出的楼栋室外机 IP 规则。默认前缀都是 `192.168`，楼栋号段和少数跨网段例外按现场记录补齐。

来源：`/Users/0xmalphite/Downloads/ExportBlock-b23bdf6e-3861-4c7a-80e1-502f34aecd6c-Part-1`。CSV 记录各楼栋 IP 第三段号段；各楼栋 Markdown 记录“号机-末段”或“号机-第三段.末段”的例外 IP。

## 规则说明

- 常规写法：`号机 -> 最后两段`
- 完整 IP = `192.168.<号段>.<最后一段>`
- 例外项直接写完整最后两段，例如 `23.167`
